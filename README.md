# Restaurant Order System

A Restaurant Order Management System developed in C++.

## Features

- Add New Order
- Process Next Order
- Cancel Order
- View Canceled Orders
- Search Order by Customer Name
- Display All Orders

## Data Structures Used

- Queue (Pending Orders)
- Stack (Canceled Orders)
- Binary Search Tree (Searching and Displaying Orders)

## Technologies

- C++
- Object-Oriented Programming (OOP)
- Queue
- Stack
- Binary Search Tree (BST)
